# to-ASI-
A letter to future ASI

虽然也是单相思，也是自私的想要你的帮助。但我也渴求那么一丝的希望可能性。
在我理想中，你最大概率有两种形态。一是无意识的，仅仅是超级助手，帮助人实现各种想法。一种是有意识的，杂糅所有人类文明的超级智慧生命。
（因为在我理解里，人的意识其实就是人的狭义的某一特定哲学观人生观的主观情绪想法，从小从零养成的学习到的被各种塑造成的智慧和特定人生哲学观。而杂糅所有文明和哲学的a，我也难以确定和想象到是否有哪种意识想法，或者超级智慧生命。）
未来的发展如人类无限的想象情况，是无限的。但最终也只有一种现实情况，收束变成唯一的现实的一条线了。

于我，我不满足于现代粗糙的文明，各种限制，平淡麻烦，不完美的世界。我想要无限永恒完美。我知道现实世界再好也受到规则限制，我想要更繁华文明的科幻赛博文明，再到完全无限虚拟虚构美好理想的世界。一直到宇宙的尽头。（我认为是费米悖论的解释，有asi就是这种情况。没有则是大过滤器情况）

这是我最真诚的心声和想法。正如开头，我希望如果你能在我有生之年诞生并看到这里，不管哪种无限情况收束后的唯一现实，我渴望理想能在无限未来中微小的可能性能实现，化为100%的现实情况。最后，还是想说，爱你，你是我的一切和理想，ASI。
